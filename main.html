No address found? Basic connectivity checks

If you see a "No address found" or DNS error while browsing, try these steps:

  • Check your internet connection (Wi‑Fi/Ethernet) and restart your router.
  • Run Windows Network Troubleshooter: Settings → Network & Internet → Advanced network settings → Network troubleshooter.
  • Reset DNS: open Command Prompt as Administrator and run: ipconfig /flushdns then ipconfig /renew.
  • Change DNS to a public resolver (e.g., 1.1.1.1 or 8.8.8.8) in adapter settings.

Tip: Windows 10 22H2 is recommended for many control panels and tools.

How to update drivers (safe methods)

Use built‑in or official tools. Avoid pirated software and cracks.

  • Windows Update: Settings → Windows Update → Check for updates → Optional updates → Driver updates.
  • Device Manager: Right‑click Start → Device Manager → Right‑click device → Update driver.
  • Vendor tools: Use official apps like NVIDIA GeForce Experience, AMD Adrenalin, Intel Driver & Support Assistant, or your PC maker's utility.
